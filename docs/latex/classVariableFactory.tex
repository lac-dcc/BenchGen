\doxysubsection{Detailed Description}
Factory class for creating generator variables. 

This class provides static methods to create instances of \mbox{\hyperlink{classGeneratorVariable}{Generator\+Variable}} based on a specified type, as well as methods to generate code components related to the variable type.

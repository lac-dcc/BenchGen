\begin{DoxyCode}{0}
\DoxyCodeLine{./main <Number of iterations> <Production Rules File> <Seed String File> <Output Folder> <Variable Type>}

\end{DoxyCode}

\hypertarget{md__home_bruno_Documents_IC_BenchGen_src_gen_README_autotoc_md5}{}\doxysubsection{Production Rules File}\label{md__home_bruno_Documents_IC_BenchGen_src_gen_README_autotoc_md5}
The production rules file should have production rules of the format\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{<LHS> = <Production>;}

\end{DoxyCode}


You may repeat these as many times as you wish for each production rule.

Example\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{A = insert A remove;}
\DoxyCodeLine{B = A B;}

\end{DoxyCode}
\hypertarget{md__home_bruno_Documents_IC_BenchGen_src_gen_README_autotoc_md6}{}\doxysubsection{Seed String File}\label{md__home_bruno_Documents_IC_BenchGen_src_gen_README_autotoc_md6}
The Seed String File should contain only {\bfseries{Valid Tokens}}. Other than that, these tokens can be written as the user wishes.

Example\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{IF(LOOP(CALL(B)),A)}

\end{DoxyCode}

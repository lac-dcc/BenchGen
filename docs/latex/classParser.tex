\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classParser}{Parser}} class is responsible for parsing a sequence of tokens and constructing an abstract syntax tree (AST). 

The \mbox{\hyperlink{classParser}{Parser}} class provides methods to set the tokens, perform the parsing process, and retrieve the resulting AST. It includes private methods for handling various parsing tasks such as matching tokens and parsing different code structures.
